## 문제 
대문자와 소문자가 같이 존재하는 문자열을 입력받아 대문자로 모두 통일하여 문자열을 출력 하는 프로그램을 작성하세요.

## 로직

```js
function solution(s){         
    var answer="";
    for(var x of s){
        var num=x.charCodeAt();
        if(num>=97 && num<=122) answer+=String.fromCharCode(num-32);
        else answer+=x;
    }

    return answer;

}

let str="ItisTimeToStudy";
solution(str);
```
