## 문제 
N개의 문자열이 입력되면 중복된 문자열은 제거하고 출력하는 프로그램을 작성하세요.

## 로직

```js
function solution(strArr){  
    var answer = [];
    
    answer = strArr.filter((v, i) => {
        if(strArr.indexOf(v)===i) 
        
        return v;
    });
    
    return answer;
}

solution(["good", "time", "good", "time", "student"]);
```
