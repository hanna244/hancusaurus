## 문제 
소문자로 된 한개의 문자열이 입력되면 중복된 문자를 제거하고 출력하는 프로그램을 작성하세요.

## 로직

```js
function solution(s){  
    var answer = "";

    for(var i = 0; i < s.length; i++){
        if(s.indexOf(s[i])===i) answer+=s[i];
    }

    return answer;
}

solution("ksekkset");
```
