## 문제 
자연수 N이 입력되면 1부터 N까지의 합을 출력하는 프로그램을 작성하세요.

## 로직

```js
function solution(arr) {
  var answer = 0 

  arr.forEach((el, item) => {
    answer = answer + item
  });

  return answer
}

solution([3, 4, 7, 9, 4, 1, 2])
```
