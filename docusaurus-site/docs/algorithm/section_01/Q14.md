## 문제 
N개의 문자열이 입력되면 그 중 가장 긴 문자열을 출력하는 프로그램을 작성하세요.

## 로직

```js
function solution(str){  
    var answer="";
    var max=Number.MIN_SAFE_INTEGER;

    for(var x of str){
        if(x.length > max) {
            max = x.length;
            answer = x;
        }
    }

    return answer;
}

solution(["teacher", "time", "student", "beautiful", "good"]);
```
