## 문제 
대문자로 이루어진 영어단어가 입력되면 단어에 포함된 ‘A'를 모두 ’#‘으로 바꾸어 출력하는 프로그램을 작성하세요.

## 로직

```js
function solution(word){
    let answer = "";
    for(var x of word){
        if(x=='A') answer+='#';
        else answer+=x;
    }
    return answer;
}

solution("BANANA");
```
