## 문제 
시험문제의 채점 결과가 주어졌을 때, 총 점수를 계산하는 프로그램을 작성하시오.

## 로직

```js
function solution(arr){         
    var answer = 0, cnt = 0;
    for(var x of arr){
        if(x === 1) {
            cnt++;
            answer+ = cnt;
        }
        else cnt=0;
    }
        
    return answer;
}

solution([1, 0, 1, 1, 1, 0, 0, 1, 1, 0]);
```
